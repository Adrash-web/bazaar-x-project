# 🛒 Bazaar-X

Bazaar-X is a **peer-to-peer marketplace web application** where users can **buy and sell products directly without middlemen**.
It provides a clean UI, product listings, and user authentication for a smooth experience.

---

## 🚀 Features

* 🔐 User Authentication (Login / Sign Up)
* 🛍️ Buy & Sell Products
* 📦 Product Listings with Images
* 🔎 Category-based Filtering (Electronics, Fashion, Furniture, Vehicles, etc.)
* 📱 Responsive Modern UI
* 📂 Image Upload Support (Multer)
* ⚡ Fast Backend with REST APIs

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/Adrash-web/bazaar-x-project.git
cd bazaar-x-project
```

### 2️⃣ Install Dependencies

```
npm install
```

### 3️⃣ Run Server

```
node server.js
```

### 4️⃣ Open in Browser

```
http://localhost:5000
```
